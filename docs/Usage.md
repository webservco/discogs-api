# Usage

## Direct usage

`\WebServCo\DiscogsApi\ApiResponse`

`GET`
```php
$apiResponse = $this->api->get($endpoint);
```

`POST`
```php
$apiResponse = $this->api->post($endpoint, $data);
```
`$data` can be a JSON encoded string, or an array (in which case it will be encoded to JSON).
